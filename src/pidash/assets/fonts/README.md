pidash# Packaged fonts

This directory contains third-party font assets shipped with the `pidash` package.

## Included fonts

- `open-sans/`
  - Assets: `OpenSans-Bold.ttf`, `OpenSans-Regular.ttf`
  - License: SIL Open Font License 1.1 (`open-sans/LICENSE`)
  - Provenance: `open-sans/README.md`
- `weather-icons/`
  - Asset: `weathericons-regular-webfont.ttf`
  - License: SIL Open Font License 1.1 (`weather-icons/LICENSE`)
  - Provenance: `weather-icons/README.md`
